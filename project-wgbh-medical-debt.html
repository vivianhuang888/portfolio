public impact case study

Exposing Medical Debt (WGBH)

Investigative analytics pipeline for identifying predatory debt collection patterns at scale.

Duration
2025
Skills
Data Engineering, Fuzzy Matching, Record Linkage, Investigative Analytics

Overview / Description

Built a data processing workflow to reconcile fragmented court records and connect debt filings back to healthcare-related claims activity.

Objective / Success Metrics

Increase linkage accuracy, reduce manual review burden, and surface patterns suitable for publication-grade investigative reporting.

Approach

Implemented normalization, phonetic keys, and multi-stage fuzzy matching to connect entities across noisy datasets with traceable confidence scoring.

Results / Key Insights

The pipeline surfaced recurring high-volume collectors and geographic concentration patterns that were previously hidden in fragmented records.

Impact / Conclusion / Future Work

Findings strengthened evidence for public-interest reporting; future work can add court outcome classification and longitudinal monitoring.
