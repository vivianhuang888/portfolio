Exposing Medical Debt (WGBH)

An investigative data pipeline to connect fragmented court filings and reveal medical debt collection patterns.

Duration
2025
Skills
Data Engineering, Record Linkage, Fuzzy Matching, Investigative Analytics, Python

Overview / Description

Medical debt data is often split across inconsistent court and claimant records, making systemic patterns difficult to detect.

This project built a reproducible linkage workflow to identify healthcare-related filings and trace concentration by collector and geography.

Objective / Success Metrics

The goal was to produce publication-ready evidence for public-interest reporting:

  1. Accuracy: Improve entity linkage across noisy legal records.
  2. Scale: Reduce manual review needs for large filing volumes.
  3. Transparency: Keep confidence scoring and audit trails for verification.

Approach

1) Data normalization

Standardized names, addresses, and claimant text fields to reduce variation from formatting and spelling noise.

  • Applied phonetic keys and token cleanup for candidate generation.
  • Built deterministic and probabilistic matching stages.

2) Linkage and validation

  • Matching: Multi-pass fuzzy linkage with thresholded confidence bands.
  • Review: Manual adjudication on ambiguous high-impact records.

3) Investigative output

Created summary tables and visuals highlighting filing concentration, repeat actors, and regional disparity patterns.

Results / Key Insights

The pipeline surfaced repeat high-volume collectors and hotspot geographies that were not visible in raw disconnected datasets.

Key discoveryRepeat collectors

Primary valueTraceable evidence

These outputs supported clear narrative framing for investigative stories and stakeholder briefings.

Impact / Conclusion / Future Work

The project improved analytical rigor and reproducibility for journalism-focused debt investigations.

  • Impact: Enabled stronger public accountability through data-backed reporting.
  • Future work: Add case outcome tracking to study downstream legal and financial consequences.
